  • 김보현
    • 커리큘럼을 잘 따라가고, 개인 기록을 빼먹지 않고 완료했다.

    • 일주일과 하루를 계획하는 즐거움
  • 윤남규
    • 어느 정도 규칙적인 생활 패턴이 형성되었으며 과제를 밀리지 않았다.

  • 허민석
    • 오랜만에 강제적으로 공부할 수 있어서 좋았습니다. 고등학교 이후로 강제로 공부해본건 오랜만인데 적응을 잘 못할 줄 알았는데 잘됐습니다.

    개선할점"

    • 개선할점

    • 김한얼
      • 취업 관련 활동을 전혀 하지 않았다.다음주부턴 블로그를 다듬으면서 이력서, 포트폴리오를 같이 다듬을 것
    • 김보현
      • 허들에서 공부하다보니 평소와 다르게 늦게 잤는데 너무 피곤했다. 평소 내 페이스대로 해야겠다.
    • 김성주
      • 시간과 분량에 쫓겨 기대하고 계획한 대로의 학습을 수행하지 못함 -> 더 깊은 수준의 탐구와 공부한 것을 여러번 복습하는 다시보기를 수행하자
      • 일상을 더 줄여보자. -> top10에 든것처럼 더 열심히 해보자
    • 윤남규
      • 수요일 4시 취침으로 수면 패턴이 깨지면서, 결국 금요일 11시 기상(..)을 해버렸다. 불규칙적인 늦은 취침 고치기!
      • Zoom, Slack을 오랜만에 사용해서 생긴 미숙함도 개선해야 한다.
    • 정수현
      • 시간 관리 미숙, 강의 부채 1번 -> 오전에 하루치 강의, 퀴즈 완료, 17시 이후에 과제 수행.
      • 회고록 미루기 -> 자기 전 일기 쓰듯이 작성
    • 허민석
      • 생각보다 시간분배가 힘들었습니다. 강의시간이 짧아서 일주일 안에 금방 들을 수 있을 것 같았는데 과제가 금요일 14시까지니까 사실상 4일안에 끝내야했는데 5일로 계산해서 마지막날에 빠듯했습니다.
      • 다음 주 부터는 4일기준으로 강의와 과제를 마무리하고 마지막 날에는 회고와 추가적인 공부를 하는 방향으로 해야겠습니다

      • 김성주
        • 매일 개인 회고 열심히 하기
        • 깃 블로그 만들고 꾸며보기
        • 알고리즘 하루에 한 문제씩 풀기
        • 다시 운동 시작하기

      • 정수현
        • 4일 강의, 퀴즈, 기본 과제 + 1일 자습/심화 과제
        • 포폴 - 깃허브 블로그 꾸미기
        • 선대, 통계학 강의 수강
      • 허민석
        • 알고리즘 문제를 꾸준히 풀어보아야겠습니다. 매번 오랜만에 푸는 느낌이 들어서 이번에 풀때는 새로 배운 개념이나 이런 것들을 정리하면서 진행해 볼 예정입니다.

        키워드"

        • 개선할점

        • 김한얼
          • LLM의 작동 원리를 이해했으며, 비지도 학습이 간단한 언어 분야와 달리, 다른 분야에서는 어떤 식으로 비지도학습을 하는지 궁금하다.
          • 다양한 GNN 종류를 이해했으며, 최근에는 어떤 모델이 쓰이는지 궁금하다.
        • 김보현
          • 나도 모르게 강의를 듣다가 딴 생각을 해서 뒤로 돌려본 적이 많았는데 집중력을 기르기 위해 아침에 일어나서 독서하는 시간을 가져야겠다.
        • 김성주
          • 깃허브 너무 겁먹었던거 같다.
          • 매일 정리하고 돌아보는 습관을 갖자.
          • 파이토치 생각보다 많이 편리하고 강했다.
        • 윤남규
          • Pytorch를 통해서 그 동안 배웠던 것들을 구현하는 과정이 꽤 재밌었고, 다양한 사람들을 만나 이야기를 들을 수 있어서 의미있는 시간을 보낸 것 같다.
        • 정수현
          • 코랩 단축키(alt+shift+L), 텐서 기본(조작, 연산) 능력 부족, 선형 회귀, 이진 분류 모델
        • 허민석
          • Pytorch와 Tensor에 대해 많이 배울 수 있었습니다. 단기간에 많은 양을 배워 응용에는 아직 익숙치 않지만 반복 학습하다보면 적응 될 것 같습니다.
          • 간단하지만 회귀모델과 분류모델을 구현해보았습니다.
